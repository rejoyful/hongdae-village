# 홍대마을 (Hongdae Village)

서울 홍대입구역 일대를 배경으로 한 픽셀아트 힐링 멀티플레이 웹 게임 (최대 10인).
각자의 공간을 꾸미고, 아이템을 만들고 거래하며, 작은 사회를 만들어갑니다.

- 디자인 스펙: `docs/superpowers/specs/2026-07-08-hongdae-village-design.md`
- 구현 계획: `docs/superpowers/plans/`

## 개발

```bash
npm install
npm run dev        # http://localhost:5173
npm test           # 단위 테스트
npm run typecheck  # 타입 검사
```

아이소메트릭 전환 수직 슬라이스는 개발 서버에서 아래 주소로 확인할 수 있습니다.
월드의 플레이어·주민·원격 캐릭터와 코디·사진·프로필 미리보기는 모두 같은 네이티브
32×48 고밀도 픽셀 렌더러를 사용하며, 기존 외형 저장값과 619조 코디 조합을 그대로 유지합니다.

```text
http://localhost:5173/?iso&offline
```

모바일에서는 화면의 빈 곳을 누른 채 끌면 손가락을 놓은 자리에 가상 조이스틱이 나타납니다.
첫 진입 안내는 실제로 한 번 움직이거나 `알겠어요`를 누르면 닫힙니다. 설정의 `화질`은 기본적으로
실제 프레임 시간에 맞춰 고품질·균형·발열 절약을 자동 선택하며, 픽셀 해상도와 게임 규칙은 유지하고
반복 장식·그림자·전투 화면 갱신만 단계적으로 줄입니다. 개발 중 터치 안내와 반응형 화면은
`&touch-preview`를 붙여 데스크톱에서도 재현할 수 있습니다.
같은 설정의 `플레이 배려`에서는 기본·큰·가장 큰 글씨, 기본·고대비 색상, 기기 설정을 따르거나 직접
고르는 움직임 최소화를 제공합니다. 선택은 로그인과 무관하게 이 기기에 저장되고 즉시 모든 DOM UI에
반영됩니다. 움직임을 줄이면 반복 애니메이션뿐 아니라 월드 장식, 전투 화면 흔들림, 레벨업·사진 플래시도
함께 멈추며 퀘스트·보상·전투 수치에는 어떠한 차이도 생기지 않습니다.

동쪽 외곽숲 자동전투·레벨업·몬스터 연구 도감을 바로 검증하려면:

```text
http://localhost:5173/?iso&offline&hunt
```

모험 일지와 10분야 생활 숙련 수첩을 이동 없이 검증하려면:

```text
http://localhost:5173/?iso&offline&quest
```

기존 생활 기록으로 열리는 8종 골목 분위기와 2.5D 월드 날씨 연출을 검증하려면:

```text
http://localhost:5173/?iso&offline&atmosphere-ready
```

`골목 분위기 관측소`는 첫 햇살·구름·비·네온 소나기·금빛 저녁·달빛 안개·첫눈·꽃잎길을
기간 제한 없이 영구 관측합니다. 이미 쌓인 산책·주민·카페·낚시·공연·사진·정원·집·요리·펫
기록을 소급해 열며, 적용과 재생은 월드의 빛과 입자만 바꾸고 전투·경제·획득 확률에는 영향을 주지 않습니다.

현재 모험 일지는 첫걸음·일일·주민 이야기·수집·전투 숙련·생활 숙련을 포함한 536개 퀘스트와
장착형 배지를 제공합니다. 생활 숙련은 골목 탐험·패션·집꾸미기·펫·이웃·골목 생활·도시 정원·골목 요리·물정원 낚시·외곽 모험의
반복 활동을 Lv.1~10으로 기록하며, 퀘스트 완료 뒤의 행동도 평생 기록에 계속 누적됩니다.
추천 탭은 초보 흐름, 받을 보상, 오늘 할 일, 거의 끝난 목표, 이야기·수집·숙련을 섞어 제안합니다.
월드의 `새 이웃 한 걸음 안내`는 이 추천 중 첫날 취향 나침반의 현재 한 활동만 작게 꺼내 보여 줍니다.
스타일·집·동행·이웃·손끝 생활·탐험 여섯 방향을 언제든 바꿀 수 있고, 버튼 하나로 기존 안전
길찾기가 실제 장소까지 이어집니다. 안내는 즉시 접어 작은 표식으로 둘 수 있으며, 선택하지 않은 방향과
이미 기록한 행동은 사라지거나 잠기지 않습니다.
원하는 퀘스트의 `목표 추적`을 누르면 완료할 때까지 위치와 다음 행동이 월드 HUD에 고정됩니다.
아이소메트릭 월드의 `골목 퀘스트 신호`는 `◆ 추적`, `✓ 보상`, `↗ 진행`, `새 이야기` 네 상태로
활동 지점과 주민 이름표 위에 나타납니다. 536개 기록을 모두 띄우지 않고 장소별 최우선 한 건,
활동 최대 8곳·주민 최대 2명만 보여 주며, 표식을 누르면 목표 추적과 안전한 금빛 길 안내가 함께 시작됩니다.
완료한 일일 보상은 활동 장소가 아니라 중앙 모험 일지로 안내됩니다.
주민 이름표의 이야기 표식이나 주민 본인을 누르면 `골목 이야기 대화`가 열립니다. 10명은 각자의
직업·관계 서사에 맞는 말투로 새 이야기·진행·추적 상태를 설명하며, 별도 수락 절차 없이 이미 한
행동이 자동 반영됩니다. 대화에서는 선택적으로 목표를 따라가거나 해당 주민의 관계 수첩을 바로 펼칠 수 있고,
나중에 닫아도 진행·신뢰·보상 기록은 사라지지 않습니다.
새 퀘스트 완료는 `골목 기록 도착` 카드로 한 번만 알려 줍니다. 현재 장면을 막지 않는 픽셀 기록책과
분류별 도장으로 완료 목표·장착 배지·직접 이어지는 다음 이야기를 한 장에 묶고, 여러 목표가 동시에
끝나면 가장 중요한 한 건만 앞에 두어 알림이 연속으로 화면을 덮지 않습니다. 다른 씬에서 끝낸 기록도
마을에 돌아왔을 때 한 번 도착하며, 재접속 때 과거 완료 기록을 다시 쏟아 내지 않습니다.
생활 탭의 대표 전문성 세 장은 단순 전시를 넘어 18개의 `생활 시너지` 조합으로 이어집니다. 서로 다른
세 분야의 정확한 조합을 맞추면 능력치 우열 없는 고유 호칭·색·픽셀 별자리 카드·기념 표식을 영구
발견하며, 조합을 바꾸어도 도감 기록은 사라지지 않습니다. 현재 시너지는 마을 명함에도 함께 공개됩니다.
주민 수첩의 `최애 팬북`에서는 만난 주민 중 최애·차애·마음친구 세 자리를 자유롭게 고르고, 신뢰
15·30·50·80·100마다 주민별 영구 응원 리본을 모읍니다. 응원석 순서를 바꾸거나 주민을 내려도
신뢰·이야기·리본은 사라지지 않으며 순위 경쟁, 선물 소비, 기간 한정 보상은 없습니다.
모험 일지의 `골목 최애 굿즈 공방`은 현재 내 캐릭터, 활성 동행 펫, 한 번 이상 만난 주민을
아크릴 스탠드·포토카드·캔 배지·키링·미니 포스터·티켓북으로 직접 제작합니다. 심야 네온·포근한 방·
옥상 초록·비 갠 물결·골목 축제·외곽 별빛의 여섯 분위기를 자유롭게 조합하고, 제작 순간의 외형과
장식을 독립 픽셀 스냅샷으로 열두 칸에 보존합니다. 대표 굿즈 세 점은 언제든 교체할 수 있고,
기존 칸 덮어쓰기는 두 번 확인하며 과거에 발견한 영구 디자인 도감은 줄어들지 않습니다.
뽑기·재료·실패·기간 제한 없이 여섯 전용 퀘스트와 마을 XP에 이어집니다.
공방에서 대표로 고른 세 점은 내 집의 `나의 최애 방 전시 코너`에 자동으로 이어집니다. 원목 선반·
타공판·유리 장식장·팬 월의 네 진열 방식과 크림·네온·낮빛·별빛의 네 조명을 자유롭게 조합하며,
고밀도 2.5D 픽셀 미리보기뿐 아니라 실제 방 벽면에도 즉시 렌더됩니다. 전시는 가구 칸과 충돌 공간을
차지하지 않고 언제든 숨길 수 있으며, 대표 굿즈와 발견 기록은 그대로 보존됩니다. 여섯 전용 퀘스트와
공간 꾸미기 성장 기록이 수집에서 생활 공간까지 이어지는 장기 루프를 만듭니다.
같은 수첩의 `작은 방 정리`에서는 주민 10명의 생활 단서를 읽어 방마다 네 물건의 제자리를 찾습니다.
오배치 감점·시간 제한·강제 순서 없이 40점을 정리하며, 완성된 2.5D 픽셀 방은 주민별 다음 장면과
기념품으로 영구 보존됩니다. 열 사람의 기념품은 전용 픽셀 보관함에서 사연을 다시 읽고 세 점까지 대표로
진열할 수 있습니다. 첫 물건부터 열 방 전집까지 7개 기록이 선택형 여정과 마을 XP에 이어집니다.

내 집 꾸미기의 `홈 스튜디오 엽서`는 현재 캐릭터 외형·동행 펫·실제 가구 좌표를 한 시점의 독립
스냅샷으로 묶어 여섯 가지 2.5D 픽셀 연출에 보존합니다. 여섯 슬롯과 대표 공개 세 칸은 순위나
촬영 연속 보상 없이 자유롭게 교체하며, 덮어쓰기·삭제는 한 번 더 확인합니다. 공개 명함에는 선택한
대표 장면만 전달되고 저장 시각과 펫 이름은 제외됩니다. 개발 화면은 아래 주소로 바로 확인합니다.

```text
http://localhost:5173/?room=1&offline&diorama-ready&home-studio
```

같은 집 메뉴의 `물건의 속삭임 도감`은 가구 카탈로그 65점 전부에 서로 다른 기억, 배치 노트,
세 가지 취향 태그를 제공합니다. 보유하거나 실제 방에 놓아 본 물건은 자동으로 발견되며, 아홉
테마 장은 각 장 네 점부터 완성되어 특정 구매나 완벽한 수집을 강요하지 않습니다. 마음에 남은
아홉 점을 최애 선반에 자유롭게 올릴 수 있고, 가구함에서는 포인터와 키보드 포커스만으로도 개별
속삭임을 바로 읽을 수 있습니다. 개발 화면은 아래 주소로 엽니다.

```text
http://localhost:5173/?room=1&offline&diorama-ready&object-stories
```

전투를 강제하지 않는 통합 마을 레벨과 8장짜리 선택형 여정은 다음 주소에서 바로 확인합니다.

```text
http://localhost:5173/?iso&offline&quest&journey-ready
```

각 여정 장은 4~10개 활동 중 3~5개만 골라 완료하면 다음 장이 열립니다. 잠긴 장의 활동을 먼저 해도
나중에 소급 인정되며 실패·기한·진행 감소가 없습니다. 모든 생활 숙련 XP와 영구 퀘스트 완료는 마을
Lv.1~50에 합산되고, 8개 여정 도장과 레벨 5·10·20·30·40·50 명찰은 실제 장착형 배지로 남습니다.
여섯 명찰은 각각 전용 코디 한 벌, 펫 픽셀 장식 하나, 가구 리폼 레시피 하나를 함께 자동 수령시키며
여정 탭의 보상 수첩에서 실제 픽셀 미리보기, 다음 보상과 사용 위치를 확인할 수 있습니다. 가이드북의
첫 화면은 패션·집 꾸미기·동행·정원·요리·낚시·이웃·탐험 8분야 평생 기록을 ‘취향 프로필’로 모읍니다.
Lv.5·10·20·30·40·50에서는 그때의 성장을 스타일·집·동행·이웃·생활·탐험 중 한 장면으로 고르는
영구 레벨업 앨범이 열립니다. 현재 캐릭터와 펫이 등장하는 360×190 아이소메트릭 장면이며, 선택은 무료로
바꿀 수 있고 자동 지급되는 코디·펫·집 보상이나 능력치에는 어떤 제한도 만들지 않습니다.
진로 탭은 스타일·집·동행·이웃·생활·탐험·수집·생태의 여덟 장기 진로를 동시에 진행하는 멀티 진로
패스포트입니다. 관심 진로 3개와 대표 진로 1개만 책갈피로 저장할 뿐 다른 진로는 잠기지 않으며, 각
진로의 이야기·수집·숙련 여섯 장을 완주하면 실제 영웅 배지와 전용 프로필 한마디가 소급 해금됩니다.
선택한 진로의 360×180 아이소메트릭 아지트에는 현재 캐릭터·동행 펫·완료 기록과 분야별 소품이 반영됩니다.
개발 모드에서는 `?iso&offline&quest&adventure-path-ready`로 아틀리에 완주 장면을 바로 확인할 수 있습니다.
가구·몬스터·취향 기록·캐릭터 스타일·동행 중 원하는 목표는 사용자별 8칸 ‘나의 수집 선반’에 담을 수
있습니다. 선반은 실제 획득 장소와 진행도를 표시하고 과거 기록도 소급 인정하며, 목표를 빼도 수집 기록은
유지됩니다. 캐릭터 코디와 펫도 픽셀 미리보기로 비교하고 아틀리에·펫샵까지 바로 길 안내를 받을 수 있습니다.
살림 쇼룸 65종은 기본 진열 37종, 4주 동안 매주 4점씩 돌아오는 큐레이션 16종, 보유 가구를
재료로 다시 만드는 DIY 12종으로 나뉩니다. 놓친 주간 가구는 반드시 다시 돌아오고 구매·판매 뒤에도
수집 이력이 유지됩니다. DIY의 재료 차감·공임 결제·완성품 지급은 서버 트랜잭션 한 번으로 처리합니다.

다른 장면·가구 도감·누적 저장은 7개 장기 퀘스트와 공간 꾸미기 숙련·선택형 여정·마을 XP로 이어집니다.

퀘스트와 마을 XP에 연결됩니다. 개발 화면은 다음 주소에서 바로 확인합니다.

화분·비 오는 처마·작은 무대·별빛 숲 여섯 장면 중 하나만 고르며, 양쪽 명함에 공개된 종·성격·장식만
360×180 2.5D 픽셀 장면에 합성합니다. 펫 별명·자유문구·코인·능력치·랭킹은 전송하지 않고 같은

않습니다. 7개 영구 퀘스트가 동행 소셜 기록을 하우징 성장으로 연결합니다.

집만 사용자별 투어 수첩에 남으며 같은 집은 서울 날짜 기준 하루 한 번만 누적됩니다. 공개 가구 구성은
카탈로그 ID 최대 18종으로만 축약해 테마별 고밀도 픽셀 투어 엽서로 복원하고, 좋아하는 여섯 장에는 개인
최애 금박을 붙일 수 있습니다. 별점·순위·연속 방문 보상은 없고, 이웃집·테마·주거형·최애 엽서 수집은
7개 장기 퀘스트와 공간 꾸미기·이웃 관계 숙련에 연결됩니다.

만료·순위는 없으며 방문·찜·분류·물건·교환 기록은 8개 퀘스트와 이웃 관계 숙련·마을 여정에 연결됩니다.

## 멀티플레이 설정 (Supabase)

1. [supabase.com](https://supabase.com)에서 프로젝트 생성
2. Authentication → Sign In / Up → **Anonymous Sign-Ins 활성화**
3. SQL Editor에서 `supabase/migrations/0001_profiles.sql`부터 `0022_neighborhood_market.sql`까지 번호순 실행
4. 프로젝트 루트에 `.env.local` 작성:

```bash
VITE_SUPABASE_URL=https://xxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJ...
```

`.env.local`이 없으면 **오프라인 혼자 모드**로 실행됩니다 (게임은 그대로 동작).
