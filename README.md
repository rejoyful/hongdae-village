# 디자이너가 만드는 세상

실제 디자이너의 요구를 하나씩 커밋하며 키워가는 아이소메트릭 웹 게임이다.

현재 버전은 `0.0`이다.

- 로그인 키아트
- 디자이너 이름 입력
- 빈 아이소메트릭 바닥
- 첫 번째 디자이너 캐릭터
- 클릭·터치·방향키 이동

기존 홍대마을의 퀘스트, 경제, 상점, 전투, 주거, 펫, 월드 콘텐츠와
서버 스키마는 제거했다. 이전 상태는 Git 이력에서만 복구할 수 있다.

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm test
npm run build
```

시각 기준은 [`docs/VISUAL_DIRECTION.md`](docs/VISUAL_DIRECTION.md)에서 관리한다.
